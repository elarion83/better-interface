Modernise l'interface administrateur WordPress avec 2 modes de design différents pour une expérience utilisateur améliorée.

== Description ==

WP Admin UI est un plugin WordPress premium qui transforme votre interface administrateur avec deux modes de design distincts, offrant une expérience moderne et personnalisée.

= Fonctionnalités principales =

* **2 Modes de Design** : Choisissez entre le mode Classique (améliorations subtiles) et le mode Moderne (design contemporain avec animations)
* **Interface Responsive** : Optimisée pour tous les appareils (mobile-first)
* **Accessibilité** : Conforme aux normes WCAG et W3C
* **Performance** : Code optimisé et léger pour un chargement rapide
* **Sécurité** : Validation et sanitisation complète des données
* **Compatibilité** : Compatible avec WordPress 5.0+ et PHP 7.4+
* **Thèmes de couleurs** : Plusieurs palettes de couleurs disponibles pour le mode moderne
* **Support multi-plugins** : Styles optimisés pour Contact Form 7, Elementor, WooCommerce et plus encore

= Mode Classique =

Le mode Classique apporte des améliorations subtiles à l'interface WordPress traditionnelle :
* Meilleure lisibilité et espacement
* Couleurs et contrastes optimisés
* Interface familière avec des raffinements discrets

= Mode Moderne =

Le mode Moderne offre une expérience visuelle complètement transformée :
* Design contemporain avec gradients et effets visuels
* Animations fluides et transitions élégantes
* Palette de couleurs vive et moderne
* Barre d'actions flottante
* Scrollbars personnalisées
* Notices modernisées

= Sécurité =

Le plugin implémente les meilleures pratiques de sécurité WordPress :
* Validation des nonces pour toutes les requêtes AJAX
* Vérification des permissions utilisateur
* Sanitisation complète des données d'entrée
* Protection contre les injections SQL et XSS

= Performance =

* Chargement conditionnel des assets selon le mode sélectionné
* CSS et JavaScript optimisés
* Cache des options WordPress
* Requêtes AJAX optimisées

== Installation ==

= Installation via Freemius =

1. Achetez et téléchargez le plugin depuis votre compte Freemius
2. Connectez-vous à votre administration WordPress
3. Allez dans **Extensions > Ajouter**
4. Cliquez sur **Téléverser une extension**
5. Sélectionnez le fichier ZIP du plugin
6. Cliquez sur **Installer maintenant**
7. Activez le plugin

= Installation manuelle =

1. Téléchargez le fichier ZIP du plugin
2. Extrayez le contenu dans le dossier `/wp-content/plugins/`
3. Renommez le dossier en `wp-admin-ui` si nécessaire
4. Activez le plugin via le menu **Extensions** dans WordPress
5. Accédez à **Outils > WP Admin UI** pour configurer le plugin

= Configuration initiale =

1. Après activation, allez dans **Outils > WP Admin UI**
2. Choisissez votre mode de design préféré (Classique ou Moderne)
3. Cliquez sur **Appliquer les changements**
4. L'interface se met à jour automatiquement

== Frequently Asked Questions ==

= Le plugin est-il compatible avec tous les thèmes ? =

Oui, WP Admin UI modifie uniquement l'interface administrateur WordPress et n'affecte pas le front-end de votre site. Il est compatible avec tous les thèmes.

= Puis-je personnaliser les couleurs ? =

Oui, le mode Moderne propose plusieurs thèmes de couleurs prédéfinis. Vous pouvez également ajouter du CSS personnalisé si nécessaire.

= Le plugin affecte-t-il les performances ? =

Non, le plugin est optimisé pour la performance. Les assets sont chargés de manière conditionnelle selon le mode sélectionné, minimisant l'impact sur les performances.

= Puis-je revenir au design WordPress par défaut ? =

Oui, vous pouvez basculer vers le mode Classique ou désactiver le plugin à tout moment. Aucune modification permanente n'est apportée à WordPress.

= Le plugin est-il compatible avec les autres plugins d'administration ? =

Oui, WP Admin UI est conçu pour être compatible avec la plupart des plugins WordPress. Des styles spécifiques sont inclus pour Contact Form 7, Elementor et WooCommerce.

= Puis-je utiliser le plugin sur plusieurs sites ? =

Cela dépend de votre licence Freemius. Consultez les détails de votre plan pour connaître le nombre de sites autorisés.
